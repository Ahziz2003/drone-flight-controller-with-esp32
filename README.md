# drone-flight-controller-with-esp32

## Overview (English)
This project implements a custom quadcopter flight controller based on an ESP32 microcontroller
with a Python Ground Station (GCS) used for telemetry monitoring and PID tuning.

The ESP32 was chosen for its integrated WiFi capabilities, dual-core architecture,
and sufficient processing power to handle sensor fusion, PID control loops,
and real-time communication with a ground control station.

---

## Présentation (Français)
Ce projet consiste à développer un contrôleur de vol pour quadricoptère basé sur un
microcontrôleur ESP32, accompagné d’une station de contrôle au sol (GCS) en Python utilisée
pour la surveillance de la télémétrie et le réglage des paramètres PID.

L’ESP32 a été choisi pour ses capacités Wi-Fi intégrées, son architecture double cœur,
et sa puissance de calcul suffisante pour gérer la fusion des capteurs, les boucles de
contrôle PID et la communication en temps réel avec la station de contrôle au sol.
